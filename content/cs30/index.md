I named my homeserver 'cs30', inspired by the [hacked server in Mr.Robot](https://mrrobot.fandom.com/wiki/Mr._Robot#:~:text=single%20infected%20server%2C-,CS30,-.%20On%20it%2C%20he).

## Services:
- [Homarr](https://github.com/ajnart/homarr) — Server web dashboard
- [Syncthing](https://github.com/syncthing/syncthing) — Synchronizing my Obsidian Vault to my PC, Macbook and Fairphone 5
- [Docuseal](https://github.com/docusealco/docuseal) — Document signing service for clients
- Homemade downloadservice using Python, Flask and Nginx
- [n8n](https://github.com/n8n-io/n8n) — Automation (for now, mainly adding things to my Google Calendar)
- [Mealie](https://github.com/mealie-recipes/mealie) — Cooking recipes
- [Dash.](https://github.com/MauriceNino/dashdot) — Server stats
- [qBittorrent](https://github.com/qbittorrent/qBittorrent) — Downloading Linux ISO's etc.
- [NocoDB](https://github.com/nocodb/nocodb) — Database with leads, network contacts, clients, and projects (using SQLite)
- Homemade internal documentation site using HUGO
- [Paperless-ngx](https://github.com/paperless-ngx/paperless-ngx) — Document management/archive system
- [Plex](https://www.plex.tv/personal-media-server/)
